# WorldForge Wiki

WorldForge is a text RPG sandbox where you create a world, make a character, and play inside a campaign that keeps track of places, people, memories, factions, secrets, and consequences.

It is not a normal chatbot with a fantasy skin. The goal is closer to a tireless tabletop game master: the game can improvise, but it also tries to remember what happened, respect what your character can know, and let the world push back.

## What This Wiki Explains

This wiki is a map of what WorldForge can do and how the main systems work in plain language. It avoids developer details, but it does explain the machinery: how a world is created, how a player turn is judged, what gets remembered, and why the narrator is not the only thing making decisions.

## What You Can Do

- Create a campaign from a short idea, crossover, genre pitch, or original premise.
- Review the generated world before playing.
- Create or import a player character.
- Explore locations, talk to people, investigate mysteries, fight, hide, bargain, travel, or make trouble.
- Watch important NPCs and factions react over time.
- Use saves/checkpoints when a scene goes sideways.

## The Core Promise

You are one person inside a world, not the only thing that exists.

If you leave a room, the people in that room can still matter. If you lie, the game should treat it as a lie, not rewrite reality. If a secret exists, the narrator should not casually reveal it before your character has a way to learn it. If you make a risky choice, the result should become part of the campaign instead of being forgotten next turn.

That is the heart of WorldForge.
