   - Type "I search the area for anyone hiding or lurking in the shadows nearby"

   - Screenshot: sidebar showing NPC list (possibly with new NPC)

   - Type "I rest here for a while and observe what the people around me are doing"
   - Submit and wait for response
   - After turn, check sidebar NPC list -- any changes (NPCs arriving/leaving)?
   - Check narrative for mentions of NPC autonomous actions
   - Screenshot: post-tick state

6. **Turn 5 -- Final interaction:**
   - Click a quick action button if available, or type another NPC-focused action
   - Verify turn completes normally
   - Final screenshot showing full game state with NPC data
