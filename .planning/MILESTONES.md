# Milestones

## v1.0 Living Sandbox

**Shipped:** 2026-04-08  
**Scope:** 31 phases, 110 plans, 199 tasks

**Delivered**
- Deterministic gameplay runtime: prompt assembly, Oracle, typed SSE turn loop, validated Storyteller tools, HP/inventory/navigation mechanics, and story control.
- Living-world simulation: episodic memory, autonomous key NPCs, reflection/progression, faction world ticks, checkpoints, and optional image generation.
- World creation stack: research-backed DNA/scaffold generation, lore extraction/search, known-IP divergence handling, reusable worldbook library, and lore editing.
- Character stack: unified ontology, structured start conditions, canonical loadouts, persona templates, and shared player/NPC authoring flows.
- Desktop non-game product surface: routed launcher, campaign creation, review, character authoring, and browser E2E hardening.
- Gameplay reconciliation baseline: docs-wide claim register, runtime matrix, and milestone handoff for the next gameplay-focused milestone.

**Known Gaps Accepted At Closeout**
- Phase-level verification artifacts are still missing for some late documentary phases (`15`, `16`, `17`, `25`, `27`, `31`, `32`).
- Gameplay quality still needs a dedicated milestone around runtime fidelity, especially reflection, checkpoint/undo integrity, state authority, and session coupling.
