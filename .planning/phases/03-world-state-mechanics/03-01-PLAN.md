  <behavior>
    - spawn_npc: inserts into npcs table with campaignId, name, persona from tags, tier="temporary", currentLocationId resolved from locationName. Returns {success: true, result: {id, name, location}}. Error if location not found.
    - spawn_item: inserts into items table. If ownerType="character", resolves owner by name across players+npcs, sets ownerId. If ownerType="location", resolves location by name, sets locationId. Error if owner/location not found.
    - reveal_location: inserts into locations table with name, description, tags. Resolves connectedTo location by name, adds bidirectional connection (updates existing location's connectedTo array too). Error if connectedTo location not found.
    - set_condition: resolves target by name across players+npcs. Reads current HP. If mode="delta", clamps hp+delta to 0-5. If mode="absolute", clamps value to 0-5. Updates hp column. Returns {success: true, result: {entity, oldHp, newHp, isDowned: newHp===0}}.
    - transfer_item: resolves item by name in campaign. Resolves target (character or location) by name. Updates item's ownerId/locationId. Clears the other field. Returns {success: true, result: {item, target, action}}.
  </behavior>

Insert into npcs table: id=crypto.randomUUID(), campaignId, name, persona=tags.join(", "), tags=JSON.stringify(tags), tier="temporary", currentLocationId=location.id, goals='{"short_term":[],"long_term":[]}', beliefs='[]', unprocessedImportance=0, inactiveTicks=0, createdAt=Date.now().

If ownerType="location", resolve location by name. Insert into items: id=crypto.randomUUID(), campaignId, name, tags=JSON.stringify(tags), ownerId (if character) or locationId (if location).

    3. `handleRevealLocation(campaignId, args)`: Extract name (string), description (string), tags (string[]), connectedToName (string). Resolve existing location by name. Insert new location: id=crypto.randomUUID(), campaignId, name, description, tags=JSON.stringify(tags), isStarting=false, connectedTo=JSON.stringify([existingLocation.id]). Then update existing location's connectedTo to include new location's id (read, parse, push, write back).

If delta provided: newHp = clamp(currentHp + delta, 0, 5). If value provided: newHp = clamp(value, 0, 5). Update hp column. Return oldHp, newHp, isDowned flag.

    5. `handleTransferItem(campaignId, args)`: Extract itemName (string), targetName (string), targetType ("character"|"location"). Resolve item by name in items table. Resolve target. Update item's ownerId/locationId, clear the other. Error if item not found.
