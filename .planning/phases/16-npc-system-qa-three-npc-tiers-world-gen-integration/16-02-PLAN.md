Test the 3 NPC creation modes as they are used from the World Review NPC tab. Each mode sends a different API request.

**Mode 1: Describe (parse-character with role=key)**

1. Send `POST http://localhost:3001/api/worldgen/parse-character` with:
   ```json
   {
     "campaignId": "{active_campaign_id}",
     "concept": "A cunning spymaster named Vera Blackthorn who operates a network of informants across all major cities. She appears as a mild-mannered librarian but controls information flow for the Shadow Guild.",
     "role": "key"
   }
   ```
2. Verify the response contains `role: "key"` and a `npc` object with:
   - `name` matching or derived from "Vera Blackthorn"
   - `description` (or `persona`) containing spy/informant themes
   - `tags` array with relevant skill/trait tags
   - Valid structure for adding to the scaffold NPC list

**Mode 2: AI Generate (generate-character with role=key)**

1. Send `POST http://localhost:3001/api/worldgen/generate-character` with:
   ```json
   {
     "campaignId": "{active_campaign_id}",
     "role": "key"
   }
   ```
2. Verify response has `role: "key"` and `npc` object with:
   - Non-empty `name`
   - Non-empty `description`/`persona` (min 20 chars)
   - `tags` array with at least 2 tags
   - Character coherent with the campaign premise

**Mode 3: Import V2 Card (import-v2-card with role=key)**

1. Send `POST http://localhost:3001/api/worldgen/import-v2-card` with:
   ```json
   {
     "campaignId": "{active_campaign_id}",
     "cardData": {
       "spec": "chara_card_v2",
       "spec_version": "2.0",
       "data": {
         "name": "Marcus the Bold",
         "description": "A fearless knight who leads the vanguard in every battle.",
         "personality": "Brave, stubborn, loyal to a fault",
         "tags": ["Knight", "Heavy Armor", "Battle Veteran"]
       }
     },
     "role": "key"
   }
   ```
2. Verify response has `role: "key"` and `npc` object derived from the V2 card data:
   - `name` is "Marcus the Bold" or derived from it
   - Tags include knight/combat related tags
   - Description/persona incorporates the personality

**Integration check:**
- After all 3 modes, verify each returned NPC object has the same structure (consistent schema)
- Verify none of the endpoints crash or return 500 errors
- If any endpoint returns a provider configuration error (no Generator key), log as SKIP

Log all 3 mode results with full request/response payloads.
