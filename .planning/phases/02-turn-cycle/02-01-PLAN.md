    Export `executeToolCall(campaignId: string, toolName: string, args: Record<string, unknown>, tick: number): Promise<ToolResult>` where `ToolResult = { success: boolean; result?: unknown; error?: string }`.

    Entity resolution helper `resolveEntity(campaignId, entityName, entityType)`:
    - Maps entityType to Drizzle table: "player" -> `players`, "npc" -> `npcs`, "location" -> `locations`, "item" -> `items`, "faction" -> `factions`
    - Queries with `eq(table.campaignId, campaignId)` AND case-insensitive name match using `sql\`LOWER(${table.name}) = LOWER(${entityName})\``
    - Returns the row or null

    Tool handlers:

    - `set_relationship`: resolve both entityA and entityB names to IDs by searching across ALL entity tables (players, npcs, locations, factions) for each name. Use Drizzle's `onConflictDoUpdate` on the unique index to upsert. Generate UUID with `crypto.randomUUID()` for new rows. Store `tags` as JSON array `[tag]` and `reason`.
    - `add_chronicle_entry`: insert into `chronicle` table with `id: crypto.randomUUID()`, `campaignId`, `tick`, `text`, `createdAt: Date.now()`. Return success with entry ID.

    Export `async function* processTurn(options: TurnOptions): AsyncGenerator<TurnEvent>`:

    1. **Query game state** from DB: player (actorTags, currentLocationId), location (environmentTags, sceneContext). Same pattern as current `/action` route.

    2. **Call Oracle**: `await callOracle(payload, options.judgeProvider)`. Yield `{ type: "oracle_result", data: oracleResult }`.

    3. **Assemble prompt**: `await assemblePrompt({ campaignId, contextWindow, actionResult: oracleResult, embedderResult, playerAction })`.

    4. **Build Storyteller system prompt**: Use `assembled.formatted` as system message. Append outcome-specific instructions:
       - Strong Hit: "The player SUCCEEDED DECISIVELY. Narrate full success with an unexpected bonus or advantage."
       - Weak Hit: "The player SUCCEEDED WITH A COMPLICATION. Narrate success but introduce a cost, complication, or partial setback."
       - Miss: "The player FAILED. Narrate the failure with meaningful consequences -- not just 'nothing happens.'"

    5. **Get chat history**: `getChatHistory(campaignId)` for conversation context.

    6. **Persist user message**: `appendChatMessages(campaignId, [{ role: "user", content: playerAction }])`.

    7. **Create tools**: `createStorytellerTools(campaignId, currentTick)`.

    8. **Call Storyteller**: Use `streamText` from `ai` with:
       - `model`: `createModel(storytellerProvider)`
       - `system`: the built system prompt string
       - `messages`: chat history (last 20) + user message
       - `tools`: from step 7
       - `stopWhen`: `stepCountIs(2)` from `ai`
       - `temperature`: from options
       - `maxOutputTokens`: from options

    9. **Iterate `result.fullStream`**: For each part:
       - `part.type === "text-delta"`: yield `{ type: "narrative", data: { text: part.textDelta } }`
       - `part.type === "tool-result"`: If the tool is `offer_quick_actions`, yield `{ type: "quick_actions", data: part.result }`. Otherwise yield `{ type: "state_update", data: { tool: part.toolName, args: part.args, result: part.result } }`.
       - Collect all text and tool results for the summary.

    10. **After stream completes**: Persist assistant message (`appendChatMessages`), increment tick (`incrementTick`).

    11. **Yield done**: `{ type: "done", data: { tick: newTick } }`.

    12. **Post-turn callback**: If `onPostTurn` provided, call it with `TurnSummary` (fire-and-forget, catch errors).

    Import `streamText`, `tool`, `stepCountIs` from `"ai"`. Import `clamp` from `../lib/index.js`.
