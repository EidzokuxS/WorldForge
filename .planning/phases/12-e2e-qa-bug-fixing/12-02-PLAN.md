## Visual QA Rubric (score each 1-5)

1. **Layout** — Grid alignment, spacing consistency, responsive behavior, no overflow/clipping
2. **Readability** — Font sizes appropriate, line heights comfortable, text contrast sufficient
3. **Visual Hierarchy** — Clear heading levels, grouped related content, scannable structure
4. **Interactivity** — Buttons/links have hover states, focus rings visible, disabled states clear
5. **Dark Theme** — Background #0a0a0a to #1a1a1a range, no white flashes, borders subtle (zinc-800), text zinc-100/200/400
6. **Polish** — Consistent spacing, aligned elements, no orphaned labels, loading states present

Score < 3 on ANY aspect = bug that must be fixed before moving on.

## Pages to Test

| Route | Page | Key Elements |
|-------|------|-------------|
| `/` | Title Screen | Logo, New Campaign button, Load Campaign list, Delete buttons |
| `/settings` | Settings | 4 tabs: Providers, Roles, Images, Research |
| `/campaign/[id]/review` | World Review | Premise, Locations, Factions, NPCs, Lore sections |
| `/campaign/[id]/character` | Character Creation | 3 modes: Describe, AI Generate, Import V2 |
| `/game` | Game Page | 3-column: left sidebar, narrative center, right sidebar |

## Dark Theme Quality Criteria
- Background: zinc-950 (#09090b) or similar very dark
- Card/panel backgrounds: zinc-900 (#18181b) or zinc-800 (#27272a)
- Primary text: zinc-100 (#f4f4f5) or white
- Secondary text: zinc-400 (#a1a1aa)
- Borders: zinc-700/800 (#3f3f46/#27272a)
- Accent: emerald/amber/blue (project-specific)
- NO pure white backgrounds, NO light gray panels, NO low-contrast text
