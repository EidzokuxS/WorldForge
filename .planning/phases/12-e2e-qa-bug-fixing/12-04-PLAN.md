## Gameplay Architecture

The game uses SSE streaming with typed events:
- `oracle_result` — chance %, outcome tier (Strong Hit/Weak Hit/Miss), reasoning
- `text` — narrative prose from Storyteller
- `state_updates` — tool call results (tags, items, HP changes)
- `npc_actions` — NPC behaviors
- `quick_actions` — 3-5 suggested next actions

## Story Control Features
- **Retry:** Hover over last AI message → Retry button → same context, new roll + narration
- **Undo:** Hover over last AI message → Undo button → pops action+response, restores snapshot
- **Edit:** Click on AI narrative text → textarea appears → edit → Ctrl+Enter to save

## Sidebar Panels (props-driven from game page)
- Left: Player HP (1-5), inventory, character info
- Right: Location name, entities present, world info
