## WorldBook Import

WorldBook import is available on the World Review page (lore section) or via a dedicated import button. The flow:
1. Upload .json file (SillyTavern WorldBook format)
2. Preview parsed entries
3. LLM classifies entries (character, location, faction, bestiary, lore)
4. Import routes entries to DB tables (NPCs, locations, factions) or lore cards

Test data location: `X:\Models\Chars` — look for .json worldbook files

## Checkpoint System

Checkpoints are accessible from the game page via a dedicated panel/dialog:
1. Save: Creates named snapshot (state.db + vectors/ + chat history)
2. Load: Restores full state from checkpoint
3. Delete: Removes checkpoint
4. Auto-checkpoint: Triggers at HP <= 2 (tested implicitly, hard to trigger manually)

## 4 States to Check
- **Empty:** No worldbooks imported, no checkpoints saved
- **Loading:** During import classification, during checkpoint save/load
- **Error:** Invalid file format, corrupt checkpoint
- **Happy path:** Successful import and checkpoint operations
