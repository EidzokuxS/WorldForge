- Assign Generator role to use this provider
- Test connection through browser

## Campaign Creation Flow

1. Title screen → "New Campaign" button
2. Step 1: Enter premise text (e.g., "A dark fantasy world of warring kingdoms")
3. Step 2: World DNA panel — toggle/edit seed categories, or skip
4. Scaffold generation — SSE progress bar showing 5 steps (premise → locations → factions → NPCs → lore)
5. World Review page — view/edit generated content (locations, factions, NPCs, lore)
6. Character Creation page — describe/generate/import a character
7. Save character → redirects to Game page

## 4 States to Check Per Page
- **Empty:** Page before any data loaded
- **Loading:** Spinner/skeleton during API calls or generation
- **Error:** What happens if LLM call fails (disconnect network briefly or use wrong API key)
- **Happy path:** Normal successful flow
