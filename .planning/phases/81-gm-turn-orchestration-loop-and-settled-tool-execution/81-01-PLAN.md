## Tasks

0. Add Stage 0 snapshot coverage for `SceneFrame`, allowed refs/tools, model-facing packet safety, scoped forecast excerpt, and private/offscreen forecast exclusion.
1. Add RED tests for GM Read shape: situation, focal actors, scene question, action interpretation, path, rationale, evidence refs, and no concrete tool payloads.
2. Implement `runGmRead` schema/prompt with compact valid examples. GM Read max: 12 top-level fields; path enum is `direct | roll_oracle | tool_plan | combat_transition | clarification | continue`.
3. Remove or deprecate `plannedTools` from the read/decision output.
4. Preserve opening-scene world-brain behavior explicitly: either migrate it into `runGmRead` with opening tests or leave opening path unchanged with a documented follow-up.
5. Verify forbidden/offscreen refs are not exposed.
6. Test migration: enumerate surviving/superseded tests from `world-brain.test.ts`, `gm-turn-decision.test.ts`, and `turn-processor.scene-plan.test.ts`; do not blanket-delete old tests.

- Player-turn GM interpretation has one read stage.
