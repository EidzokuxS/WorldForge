  <action>
1. In `backend/src/db/index.ts`, add and export `getSqliteConnection()` that returns the raw `Database.Database` instance (or throws if not connected). This is needed for `better-sqlite3`'s `.backup()` API.

2. In `backend/src/campaign/paths.ts`, add:
   - `getCheckpointsDir(campaignId: string): string` -- returns `campaigns/{id}/checkpoints`
   - `getCheckpointDir(campaignId: string, checkpointId: string): string` -- returns `campaigns/{id}/checkpoints/{checkpointId}`

3. Create `backend/src/campaign/checkpoints.ts` with:

   **Type:** `CheckpointMeta = { id: string; name: string; description: string; createdAt: number; auto: boolean }`

   **`createCheckpoint(campaignId, opts?: { name?: string; description?: string; auto?: boolean })`:**
   - assertSafeId on campaignId
   - Generate checkpoint ID as `{Date.now()}-{name || 'manual'}` (sanitize name: replace non-word chars with `-`, truncate to 40 chars)
   - Create checkpoint dir: `campaigns/{id}/checkpoints/{checkpointId}/`
   - Use `getSqliteConnection().backup(path.join(checkpointDir, 'state.db'))` for safe SQLite backup (returns Promise)
   - Use `fs.cpSync(vectorsDir, path.join(checkpointDir, 'vectors'), { recursive: true })` to copy vectors
   - Use `fs.copyFileSync(chatHistoryPath, path.join(checkpointDir, 'chat_history.json'))` to copy chat
   - Write `meta.json` with CheckpointMeta fields
   - Return CheckpointMeta

   **`listCheckpoints(campaignId)`:**
   - Read checkpoints dir, filter directories, read each `meta.json`
   - Sort by createdAt descending (newest first)
   - Return CheckpointMeta[]

   **`loadCheckpoint(campaignId, checkpointId)`:**
   - assertSafeId on both IDs
   - Verify checkpoint dir and meta.json exist (throw 404 if not)
   - `closeDb()` to release SQLite connection
   - `closeVectorDb()` to release vector connection
   - Copy checkpoint `state.db` over campaign `state.db` using `fs.copyFileSync`
   - Copy checkpoint `vectors/` over campaign `vectors/` using `fs.rmSync` + `fs.cpSync`
   - Copy checkpoint `chat_history.json` over campaign `chat_history.json`
   - `connectDb(campaignDbPath)` to reconnect
   - `runMigrations()` in case of schema evolution
   - `await openVectorDb(campaignId)` to reconnect vectors
   - Return the loaded CheckpointMeta

   **`deleteCheckpoint(campaignId, checkpointId)`:**
   - assertSafeId on both IDs
   - Verify dir exists (throw 404 if not)
   - `fs.rmSync(checkpointDir, { recursive: true, force: true })`

   **`pruneAutoCheckpoints(campaignId, keepCount = 3)`:**
   - List checkpoints, filter `auto: true`, sort by createdAt ascending
   - Delete all but the newest `keepCount`

4. Update `backend/src/campaign/index.ts` barrel to re-export everything from checkpoints.ts.
